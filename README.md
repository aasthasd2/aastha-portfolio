# Aastha Ojha — Personal Portfolio

A colorful, single-page portfolio website built from my resume.

## Files
- `index.html` — page content
- `styles.css` — styling (colorful/bold theme)
- `script.js` — small interactions (scroll reveal, mobile menu)

## Run locally
Just open `index.html` in any browser. No build step needed.
